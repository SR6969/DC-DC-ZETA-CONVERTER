# DC-DC Zeta Converter

## Project Overview
This project presents the design, analysis, simulation, and hardware implementation of a DC-DC Zeta Converter.

The converter is capable of both step-up and step-down voltage conversion while maintaining the same output polarity as the input. It provides continuous output current, making it suitable for sensitive electronic loads.

## Specifications
- Input Voltage: 10 V
- Output Voltage: 8 V – 15 V
- Output Current: 1 A
- Switching Frequency: 7.5 kHz

## Key Features
- Continuous Conduction Mode (CCM) analysis
- Discontinuous Conduction Mode (DCM) demonstration
- Diode current and switch voltage waveforms
- LTspice simulation results
- PCB-based hardware implementation
