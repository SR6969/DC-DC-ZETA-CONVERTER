DC-DC Zeta Converter

 Project Overview

This project presents the design and implementation of a DC-DC Zeta Converter, a non-isolated power electronic converter capable of both stepping up and stepping down the input voltage while maintaining the same output polarity.

The project was carried out as part of the Electrical Machines and Power Electronics Laboratory (EMPEL). The converter was designed, analyzed, fabricated on a PCB, and experimentally verified through hardware testing.

⸻

 Objectives

* Design a DC-DC Zeta Converter.
* Obtain an adjustable output voltage from a fixed DC input.
* Study the converter in Continuous Conduction Mode (CCM).
* Demonstrate Discontinuous Conduction Mode (DCM) by varying the switching frequency.
* Observe diode current and switch voltage waveforms.
* Validate converter performance using hardware implementation.

⸻

⚙️ Specifications

Parameter	Value
Input Voltage	10 V
Output Voltage	8–15 V
Output Current	1 A
Switching Frequency	7.5 kHz

 Components Used

⸻

Methodology

1. Calculated the design parameters for the Zeta Converter.
2. Selected suitable inductors, capacitors, and switching devices.
3. Designed and fabricated the converter on a PCB.
4. Generated PWM signals using a PWM generator module.
5. Tested the converter under different duty cycles.
6. Recorded the output voltage, diode current, and switch voltage waveforms.
7. Reduced the switching frequency to observe operation in DCM.

⸻

 Results

* Successfully achieved both step-up and step-down voltage conversion.
* Verified Continuous Conduction Mode (CCM) operation.
* Demonstrated Discontinuous Conduction Mode (DCM) by reducing the switching frequency.
* Observed expected diode current and switch voltage waveforms.
* Hardware implementation produced stable output within the specified operating range.
